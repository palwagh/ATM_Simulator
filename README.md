ATM Simulator

This is a simple ATM Simulator project that allows users to perform basic banking operations. Users can check their balance, deposit money, withdraw money, and view transaction history. It is designed to mimic the basic functions of a real ATM machine.

⭐ Features

Check balance

Deposit money

Withdraw money

View transaction history

Secure PIN login

▶️ How to Run

Run the main program file (example):

python main.py
